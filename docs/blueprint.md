# **App Name**: Edoxia-TCG

## Core Features:

- Card Display: Display TCG cards with details like name, image, stats, and cost.
- Card Styling: Apply distinct styling to cards based on type.
- Interactive Hover: Implement a hover effect for an interactive user experience.
- Dynamic Card Layout: Display cards in a responsive CSS grid layout.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a mystical and magical feel.
- Background color: Light gray (#F5F5F5), almost white, to allow the colorful cards to pop.
- Accent color: Vibrant blue (#2196F3) for interactive elements and highlights.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, and 'Inter' (sans-serif) for body text to ensure readability.
- Cards are arranged on the screen using a CSS grid
- Use clear and thematic icons for mana cost, attack, and defense.
- Subtle zoom effect on hover to signal interactivity.
